build Framework Installation

I chose Materialize CSS, a modern responsive front-end framework based on Material Design principles.

Installation Steps:
  1. Created a GitHub repository named beenishsaeed26.github.io
  2. Enabled GitHub Pages in repository settings (Settings → Pages → Source: main branch)
  3. Added Materialize CSS via CDN in the HTML head:
    <link href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css" rel="stylesheet">
  4. Added Material Icons for icon support
  5. Created index.html and pushed to GitHub

Key Takeaways:

  • Responsive design is built-in, making mobile compatibility easy
